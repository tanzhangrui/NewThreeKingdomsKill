# 三国杀 · 天意侵蚀 v2.0

一款基于 Qt 5 开发的三国杀卡牌对战游戏，采用 C++11 编写，具有精美的视觉效果和完整的回合制游戏逻辑。

## v2.0 版本更新亮点

### 🎮 游戏系统全面升级

- **角色选择系统**：全新的开始菜单，可自主选择曹操、刘备、司马懿三位武将
- **阵营统一**：所有武将统一使用"邪·xxx"命名格式，体现天意侵蚀主题
- **出闪响应机制**：被敌方使用【杀】时，若有【闪】会弹出选择框，给予5秒时间决定是否使用

## 游戏玩法

### 基本规则

- 游戏开始时，选择一名武将并摸取初始手牌
- 回合流程: **摸牌阶段** → **出牌阶段** → **弃牌阶段**
- 每回合自动摸取 2 张牌
- 出牌阶段可使用手牌攻击敌人或回复自身
- 回合结束时，手牌数不能超过当前体力值
- **胜利条件**: 成为最后存活的玩家

### 操作说明

| 操作        | 说明            |
| --------- | ------------- |
| 点击武将卡     | 在角色选择界面选择武将   |
| 点击手牌      | 选中卡牌，进入目标选择模式 |
| 点击目标      | 对选中的目标使用卡牌/技能 |
| 技能按钮      | 使用当前武将的主动技能   |
| 结束回合      | 提前结束当前回合      |
| 使用/不使用【闪】 | 被攻击时选择是否出闪    |

## 卡牌系统

### 基本牌

| 卡牌    | 效果          | 备注               |
| ----- | ----------- | ---------------- |
| **杀** | 对目标造成 1 点伤害 | 每回合限用 1 张（司马懿除外） |
| **闪** | 抵消一张【杀】的伤害  | 被动使用             |
| **桃** | 恢复 1 点体力    | 不能超过体力上限         |

### 锦囊牌

| 卡牌             | 效果                   |
| -------------- | -------------------- |
| **顺手牵羊**       | 获得目标 1 张手牌           |
| **过河拆桥**       | 弃置目标 1 张手牌           |
| **一对跳跳虎两条脆脆鲨** | 全体玩家回复 1 点血量         |
| **天意侵袭**       | 全体玩家必须出【杀】，否则受 1 点伤害 |

## 核心类设计

### 类图关系

```
MainWindow (UI管理)
    │
    ├── GameEngine (游戏逻辑)
    │       │
    │       ├── Player (玩家数据)
    │       │       ├── Hero* (武将)
    │       │       └── QList<Card*> (手牌)
    │       │
    │       └── Card (卡牌) / Hero (武将)
    │
    ├── HeroPanel (武将面板组件)
    │
    ├── CenterEffectWidget (中央特效组件)
    │
    └── AnimOverlay (动画叠加层)
```

### 主要类职责

| 类                    | 文件                | 职责                  |
| -------------------- | ----------------- | ------------------- |
| `MainWindow`         | mainwindow\.h/cpp | 主窗口、UI渲染、动画系统、事件处理  |
| `GameEngine`         | gameengine.h/cpp  | 游戏逻辑、回合管理、AI决策、技能触发 |
| `Player`             | Player.h/cpp      | 玩家数据、手牌管理           |
| `Hero`               | Hero.h/cpp        | 武将基类及具体武将实现         |
| `Card`               | Card.h/cpp        | 卡牌基类及具体卡牌实现         |
| `HeroPanel`          | mainwindow\.cpp   | 武将面板UI组件            |
| `CenterEffectWidget` | mainwindow\.cpp   | 中央艺术字特效组件           |
| `AnimOverlay`        | mainwindow\.cpp   | 动画效果叠加层             |
