# 三国杀 · 天意侵蚀

一款基于 Qt 6 开发的三国杀卡牌对战游戏，采用 C++17 编写，具有精美的视觉效果和完整的回合制游戏逻辑。

## 功能特性

- **三方对弈模式**: 1名玩家 vs 2名AI对手
- **回合制战斗**: 经典的摸牌、出牌、弃牌流程
- **丰富卡牌系统**: 杀、闪、桃、锦囊牌等多种卡牌
- **武将技能系统**: 3位三国名将，各具独特技能
- **智能AI对手**: 具有策略决策能力的AI系统
- **精美视觉效果**:
  - 动态渐变背景与粒子特效
  - 卡牌飞行动画
  - 技能闪光与伤害数字浮动
  - 血量条动态变化

## 技术栈

- **框架**: Qt 6 (Widgets)
- **语言**: C++17
- **构建工具**: qmake
- **开发环境**: Qt Creator

## 游戏玩法

### 基本规则

- 游戏开始时，每位玩家获得一名武将并摸取初始手牌
- 回合流程: **摸牌阶段** → **出牌阶段** → **弃牌阶段**
- 每回合自动摸取 2 张牌
- 出牌阶段可使用手牌攻击敌人或回复自身
- 回合结束时，手牌数不能超过当前体力值
- **胜利条件**: 成为最后存活的玩家

### 操作说明

| 操作   | 说明            |
| ---- | ------------- |
| 点击手牌 | 选中卡牌，进入目标选择模式 |
| 点击目标 | 对选中的目标使用卡牌/技能 |
| 技能按钮 | 使用当前武将的技能     |
| 结束回合 | 提前结束当前回合      |
| 开始游戏 | 重新开始新一局游戏     |

## 卡牌系统

### 基本牌

| 卡牌    | 效果          | 备注        |
| ----- | ----------- | --------- |
| **杀** | 对目标造成 1 点伤害 | 每回合限用 1 张 |
| **闪** | 抵消一张【杀】的伤害  | 被动使用      |
| **桃** | 恢复 1 点体力    | 不能超过体力上限  |

### 锦囊牌

| 卡牌       | 效果         |
| -------- | ---------- |
| **顺手牵羊** | 获得目标 1 张手牌 |
| **过河拆桥** | 弃置目标 1 张手牌 |

## 核心类设计

### 类图关系

```
MainWindow (UI管理)
    │
    ├── GameEngine (游戏逻辑)
    │       │
    │       ├── Player (玩家数据)
    │       │       ├── Hero* (武将)
    │       │       └── QList<Card*> (手牌)
    │       │
    │       └── Card (卡牌) / Hero (武将)
    │
    ├── HeroPanel (武将面板组件)
    │
    └── AnimOverlay (动画叠加层)
```

### 主要类职责

| 类             | 文件                | 职责             |
| ------------- | ----------------- | -------------- |
| `MainWindow`  | mainwindow\.h/cpp | 主窗口、UI渲染、动画系统  |
| `GameEngine`  | gameengine.h/cpp  | 游戏逻辑、回合管理、AI决策 |
| `Player`      | Player.h/cpp      | 玩家数据、手牌管理      |
| `Hero`        | Hero.h/cpp        | 武将基类及具体武将实现    |
| `Card`        | Card.h/cpp        | 卡牌基类及具体卡牌实现    |
| `HeroPanel`   | mainwindow\.cpp   | 武将面板UI组件       |
| `AnimOverlay` | mainwindow\.cpp   | 动画效果叠加层        |

## 扩展开发

### 添加新武将

1. 在 `Hero.h` 中创建新的武将类继承 `Hero`
2. 实现技能方法
3. 在 `GameEngine::initGame()` 中注册新武将

### 添加新卡牌

1. 在 `Card.h` 中创建新的卡牌类继承 `Card`
2. 实现卡牌效果
3. 在卡牌工厂中注册新卡牌

### 扩展头文件

`src/` 目录下提供了更丰富的扩展设计参考:

- 更多卡牌类型 (火杀、雷杀、决斗、万箭齐发等)
- 完整的花色系统 (黑桃、红桃、梅花、方块)
- 卡牌工厂模式
