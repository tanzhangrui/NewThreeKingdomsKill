## 🎮 游戏简介

《三国杀·天意侵蚀》是一款创新的三国杀卡牌游戏，采用三方混战模式。游戏对经典三国杀武将进行了大胆的"邪化"改编，每位武将都拥有独特而强大的技能组合，带来全新的游戏体验。

### ✨ 核心特色

- 🎭 **邪化武将系统** - 三位风格迥异的邪化武将，每位都有独特的技能机制
- 🤖 **智能AI对手** - 具备策略决策能力的AI系统，提供有挑战性的对战体验
- 🎬 **技能特效视频** - 每个技能发动时播放专属特效视频，沉浸感十足
- ✨ **炫酷动画效果** - 粒子特效、卡牌飞行、伤害数字等丰富的视觉反馈
- 🎨 **精美UI设计** - 深色玄幻主题，金色点缀，视觉效果出众

## 🦸 武将介绍

### 曹操 - 邪化霸主
> *"宁教我负天下人，休教天下人负我"*

| 技能名 | 类型 | 效果 |
|--------|------|------|
| 战无不胜の斗鸡眼 | 被动 | 血量≤50%时，对全体敌方造成1点精神伤害 |
| 呱 | 被动 | 本回合未出牌时，对指定玩家造成1点精神伤害 |
| 汪 | 主动 | 与目标进行"狗叫对决"，输者受1点伤害 |
| 临终关怀 | 被动 | 死亡时，对全体玩家造成1点精神伤害 |

**定位**: 高风险高回报的"自爆型"武将，适合激进打法。

---

### 刘备 - 邪化仁君
> *"勿以恶小而为之，勿以善小而不为"*

| 技能名 | 类型 | 效果 |
|--------|------|------|
| 自刎归天 | 被动 | 打出【杀】时，全员（包括自己）受1点伤害 |
| 无敌の二弟 | 被动 | 受伤后可召唤二弟对攻击者造成1点伤害 |
| 仁之剑义之剑 | 主动 | 一名玩家回1血，另一名玩家受1点伤害 |

**定位**: "同归于尽"型武将，牺牲自己换取对敌人的伤害。

---

### 司马懿 - 邪化谋士
> *"天意难违，顺势而为"*

| 技能名 | 类型 | 效果 |
|--------|------|------|
| 天意化骨掌 | 被动 | 可无限出【杀】，每回合第二张杀开始触发特效 |
| 天意面瘫 | 被动 | 打出【桃】时，自身回2血，其他人回1血 |

**定位**: "控制型"武将，擅长持续输出和群体治疗。

## 🃏 卡牌系统

### 基础牌

| 卡牌 | 效果 | 备注 |
|------|------|------|
| 【杀】 | 对目标造成1点伤害 | 每回合限用1张（司马懿除外） |
| 【闪】 | 抵消【杀】的伤害 | 只能被动使用 |
| 【桃】 | 恢复1点体力 | 不能超过体力上限 |

### 锦囊牌

| 卡牌 | 效果 |
|------|------|
| 顺手牵羊 | 获得目标1张手牌 |
| 过河拆桥 | 弃置目标1张手牌 |
| 一对跳跳虎两条脆脆鲨 | 全体玩家回复1点血量 |
| 天意侵袭 | 全体玩家必须出【杀】，否则受到1点伤害 |

## 🚀 快速开始

### 环境要求

- **Qt 5.x** 或更高版本
- **C++11** 编译器支持
- **操作系统**: Windows / Linux / macOS

### 编译步骤

#### Windows (Qt Creator)

1. 克隆仓库
```bash
git clone https://github.com/yourusername/NewThreeKingdomsKill.git
cd NewThreeKingdomsKill
```

2. 使用Qt Creator打开 `NewThreeKingdomsKill.pro`

3. 点击 "构建" → "运行" 或按 `Ctrl+R`

#### 命令行编译

```bash
# 生成Makefile
qmake NewThreeKingdomsKill.pro

# 编译
make

# 运行
./ThreeKingdomsKill
```

### 注意事项

- 确保 `img/` 和 `video/` 目录与可执行文件在同一目录
- 视频播放需要系统安装相应的解码器（推荐 K-Lite Codec Pack）

## 🎯 游戏玩法

### 基本规则

1. **开局**: 选择一位武将，游戏开始时每位玩家摸4张牌
2. **回合流程**:
   - 摸牌阶段：摸2张牌
   - 出牌阶段：使用卡牌和技能
   - 弃牌阶段：手牌上限为当前体力值
3. **胜利条件**: 成为最后存活的玩家

### 操作说明

| 操作 | 说明 |
|------|------|
| 点击手牌 | 使用该卡牌（需要选择目标时进入目标选择模式） |
| 点击玩家面板 | 在目标选择模式下选择目标 |
| 点击技能按钮 | 发动主动技能 |
| 点击"结束回合" | 结束当前回合 |

## 🛠️ 技术架构

### 技术栈

- **开发框架**: Qt 5.x (Widgets + Multimedia)
- **编程语言**: C++11
- **构建系统**: qmake
- **核心模块**:
  - `QtCore` - 核心数据结构和事件系统
  - `QtWidgets` - GUI组件
  - `QtMultimedia` - 视频播放
  - `QtGui` - 自定义绘制和动画

### 项目结构

```
NewThreeKingdomsKill/
├── main.cpp              # 程序入口
├── mainwindow.h/cpp      # 主窗口与UI
├── gameengine.h/cpp      # 游戏引擎核心
├── Hero.h/cpp            # 武将系统
├── Card.h/cpp            # 卡牌系统
├── Player.h/cpp          # 玩家类
├── img/                  # 图片资源
└── video/                # 技能特效视频
```

